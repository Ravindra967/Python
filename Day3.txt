### ðŸ”¹ `while` loop

A `while` loop **executes repeatedly** as long as a condition is `True`.

#### ðŸ“Œ Syntax:

```python

```

#### ðŸ’¡ Example:

### ðŸ”¹ `for` loop

Used to **iterate over sequences** like lists, strings, ranges, etc.

#### ðŸ“Œ Syntax:

```python

```

#### ðŸ’¡ Example:

```python

```

### ðŸ”¹ `break`, `continue`, `pass`

These **control loop execution**.

#### âœ… `break`: Exits the loop immediately.

```python

    if i == 5:
        break
    print(i)
```

**Output:** `1 2 3 4`

---

#### âœ… `continue`: Skips current iteration, continues to next.

```python

    if i == 3:
        continue
    print(i)
```

**Output:** `1 2 4 5`

---

#### âœ… `pass`: Does nothing (acts as a placeholder).

```python

**Output:** (No output â€” it just passes.)

### ðŸ”¹ Nested Loops

A **loop inside another loop**. Often used for matrices or grids.

#### ðŸ’¡ Example:

    for j in range(1, 4):     # Inner loop
        print(f"{i} x {j} = {i * j}")
    print("-----")
```

#### âœ… Output:

```

-----

3 x 3 = 9
-----
```

> ðŸ§  Think of nested loops as rows and columns in a table or grid.
